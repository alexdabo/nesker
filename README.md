# Nesker

Nesker is a task manager

## Description

Nesker is a task manager made with Nestjs and React

## Installation

```bash
$ npm install
```

## Environment

```bash
# Database type (postgres | mssql)
DATABASE_TYPE

# Database connection string
DATABASE_URL
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
